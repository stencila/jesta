    • (update: boolean): Manifest

    • Get the manifest for this plugin.

      This function reads in the plugin's codemeta.json file and populates the featureList using the schema property of each method. This approach allows plugin developers to easily extend Jesta by writing their own codemeta.json file and method schemas.

      Parameters

      • update: boolean

        Should the codemeta.json file be updated?

      Returns Manifest
